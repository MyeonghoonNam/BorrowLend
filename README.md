# 프로젝트 소개

중고물건 대여 서비스를 제공하며 다양한 기기의 제한 없이 이용 가능한 반응형 웹
<br><br>

### 개발환경

클라이언트 : Html, CSS, JavaScript<br>
서버 : NodeJS<br>
DBMS : MongoDB<br><br>

# 주요기능

#### 물품 검색

- 등록된 물품에 관한 검색

#### 게시판 생성/조회/수정/삭제

- 물품, 고객지원, 쪽지함의 생성, 삭제, 수정, 조회 기능별 구현
- 이미지 슬라이더 및 모달 구현

#### 실시간 비동기 서비스

- 쪽지함 전체 기능
- 좋아요 기능
- 댓글 기능
